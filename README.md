# mcdiceface

A silly web game for smart phones. You can find a [live demo here](https://mcdiceface.netlify.com).

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
